Description
===========

jbiot J. bioinformatic devtoolkits which contain::

    - lsub
    - csub 
    - smartFqs.py
    - jbio.start/stop/send
    - docker2sing.py
    - startup
    - md2html.py
    - always 
    - filterTable.py
    - render.py
    - tsv2xls.py
    - xls2tsv.py


install it and try them::
    
    pip install jbiot
