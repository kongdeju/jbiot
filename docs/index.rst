Installation
============
use pip to install::


    pip intall jbiot

or use git to clone latest code::
    
    git clone git@123.57.226.13:/expan/DevRepos/jbiot.git


with jbiot install on your computer, the following script can be used::

    - lsub
    - csub 
    - smartFqs.py
    - jbio.start/stop/send
    - docker2sing.py
    - startup
    - md2html.py
    - always 
    - filterTable.py
    - render.py
